# 身體作為訊息的物質前提：邊界、介面與路徑的創作研究

陳羿言 · 國立清華大學科技藝術研究所 · 碩士學位口試簡報（2026.07）

**線上瀏覽：https://cyiyen.github.io/beacon-silence-defense/**

本研究透過三件互動裝置——《Solitude Internet》（邊界）、《紙殼迴聲》（介面）、《Beacon Silence》（路徑）——探討觀眾的身體如何成為訊息傳遞的物質前提。核心作品《Beacon Silence》以 Li-Fi 雷射光束傳輸七聲道聲音，觀眾身體遮蔽光束即切斷對應聲道；當所有光束同時被身體阻斷，聲景歸於「集體寂靜」。

## 使用說明

（簡報右下角的「?」按鈕也有同樣的說明）

### 總覽（球形）
- 左鍵點卡片進入該頁；右鍵拖曳旋轉球體；按數字鍵跳到對應頁

### 簡報瀏覽
- `←` `→` / 空白鍵：翻頁
- 數字鍵：跳頁；`Esc`：回總覽

### 影片頁（Solitude Internet／紙殼迴聲）
- `R`：重播；`S`：顯示進度條（拖曳可跳轉）

### 即時演示 · 七光束
- 左鍵拖動觀眾去擋光束；中鍵旋轉視角；右鍵平移；滾輪縮放
- 「♪ 開啟聲音」：播放七聲道（光束被遮住，對應聲道即中斷）
- 「▶ 原曲」：播放完整原曲

### 演講者模式
- `P`：開啟第二視窗（逐字稿＋計時器＋雙向同步翻頁）
- 稿子可直接點擊編輯（自動儲存於瀏覽器）；「還原此頁」單擊還原本頁、雙擊全部還原

### 顯示模式
- `B`：切換「滿版高對比」／「電影感」（設定會記住）

## 建議環境

桌面版 Chrome / Edge / Safari；首次載入含影片約 140MB，建議 Wi-Fi。
